## 1. 认证方式

系统支持以下认证方式，覆盖不同场景：

| 方式 | 适用场景 | 便捷度 | 安全等级 |
|------|----------|--------|----------|
| 邮箱 + 密码 | Web 主站注册 | 中 | 高 (bcrypt + 可选 TOTP) |
| GitHub OAuth | 开发者快捷登录 | 高 | 高 (OAuth2 + PKCE) |
| Google OAuth | 通用快捷登录 | 高 | 高 (OAuth2 + PKCE) |
| API Key | SDK / CLI / 自动化脚本 | — | 高 (HMAC-SHA256 签名) |
| 个人访问令牌 (PAT) | 第三方集成、CI/CD | — | 高 (细粒度权限 + 可撤销) |

## 2. 用户注册流程

### 2.1 邮箱注册

```
访问注册页 → 输入邮箱 + 密码 → 邮箱验证（发送验证链接）
→ 点击验证链接 → 选择用户名 → 完成注册 → 自动登录
```

- 密码强度要求：最少 8 位，包含大小写字母 + 数字
- 邮箱验证链接 24 小时有效
- 注册即创建个人默认租户

### 2.2 OAuth 注册

```
点击 "使用 GitHub / Google 登录" → 跳转 OAuth 授权页
→ 授权 → 回调 → 首次登录自动创建账号 → 进入引导流程
```

- 首次 OAuth 登录需补充用户名
- OAuth 账号与邮箱账号自动关联（同邮箱）

### 2.3 手机号注册（中国区）

```
访客点击[注册]
    ↓
选择注册方式
    ├── 邮箱注册 → 填写邮箱/密码 → 验证邮件 → 完善资料
    ├── OAuth (Google/GitHub) → 授权 → 绑定邮箱 → 完善资料
    └── 手机号注册（中国区）→ 输入手机号 → 短信验证码 → 设置密码 → 完善资料
    ↓
选择角色标签（可多选）：首席研究员 / 领域建模师 / 数据分析师 / 策略验证员 / 学生
    ↓
进入引导式创建首个项目
```

**手机号注册流程**：
1. 用户输入中国大陆手机号（11位，1[3-9]开头）
2. 点击"获取验证码"，后端调用短信渠道发送 6 位数字验证码（5 分钟有效）
3. 用户输入验证码，后端校验
4. 设置密码（≥8位，含大小写字母+数字）
5. 完善资料：用户名（2-30字符）、选择角色标签
6. 注册即创建个人默认租户，自动登录

**安全限制**：
- 同一手机号每日最多发送 5 条验证码
- 同一 IP 每小时最多发送 10 条验证码
- 验证码错误 3 次后需等待 60 秒重新获取

### 2.4 渐进式认证

| 状态 | 可访问内容 | 触发登录墙 |
|------|-----------|-----------|
| 未登录访客 | 浏览公开项目、查看社区广场 | 执行仿真、创建项目、使用科研智能体 |
| 已登录免费用户 | 个人工作台、社区互动 | 超出免费配额的操作 |

## 3. 登录流程

```
输入凭证 → 认证 → JWT 签发 → 前端存储 token → 后续请求携带 Authorization Header
```

- JWT 有效期：Access Token 2 小时，Refresh Token 30 天
- 支持"记住我"（延长 Refresh Token 至 90 天）
- 登录失败 5 次后触发验证码（30 分钟内）
- 异地登录检测与告警邮件

## 4. 密码管理

| 功能 | 说明 |
|------|------|
| 忘记密码 | 输入邮箱 → 发送重置链接（1 小时有效）→ 设置新密码 |
| 修改密码 | 需输入当前密码验证 |
| 密码历史 | 不可与最近 3 次密码相同 |
| 强制修改 | 管理员可强制用户下次登录修改密码 |

## 5. API Key 管理

用户可在个人设置中创建和管理 API Key：

| 操作 | 说明 |
|------|------|
| 创建 Key | 指定名称、权限范围（read/write/admin）、速率限制 |
| 查看 Key | 仅创建时完整显示一次，之后只显示前缀 |
| 撤销 Key | 立即生效，不可恢复 |
| 使用统计 | 查看最近 30 天调用次数 |

API Key 权限范围：
- `read` — 只读操作（查询项目、获取结果）
- `read:write` — 读写操作（创建项目、启动仿真）
- `read:write:admin` — 全权限（等同于 Web 端操作）

## 6. 安全要求

| 要求 | 说明 |
|------|------|
| 密码存储 | bcrypt，cost ≥ 12 |
| JWT 签名 | RS256 非对称算法 |
| 传输安全 | 全站 HTTPS，HSTS 启用 |
| 会话管理 | 无状态 JWT，服务器不存储会话 |
| 登出 | 客户端清除 token，无需服务端操作 |
| 多因素认证 | TOTP 可选，管理员强制要求 |
| 异常检测 | 异地登录告警、频繁失败限流 |
