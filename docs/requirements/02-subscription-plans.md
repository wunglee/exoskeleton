## 3. RP（Research Points）积分体系

### 3.1 核心设计

RP 是平台内统一消费度量，弥合订阅制（Stripe）和一次性支付（支付宝/微信）的差距：

```
汇率锚定：1 CNY = 10 RP | 1 USD = 70 RP
最低购买：1,000 RP (¥100 / ~$14)
每月赠送：免费版 100 RP | 专业版 1,000 RP | 团队版 5,000 RP | 企业版 50,000 RP
```

### 3.2 RP 获取方式

| 维度 | 指标 | 积分 |
|------|------|------|
| **购买** | 直接购买 RP 包 | 按汇率 |
| **月度配额** | 订阅计划每月赠送 | 按计划 |
| **公众评价** | 星标 | +10 RP/星（每个用户仅计一次） |
| | 高质量评论 | +30 RP/条（≥100字且被 3+ 点赞） |
| **关注度** | 浏览量 | +0.01 RP/次（UV 去重） |
| | 被 Fork | +50 RP/Fork |
| | 被打开运行 | +20 RP/次 |
| **实际贡献** | 合并请求被接受 | +250~1000 RP（按影响权重） |
| | 发布首个公开项目 | +200 RP（一次性） |

### 3.3 RP 消耗

| 操作 | RP 消耗 |
|------|---------|
| 因果发现任务 | 10 RP/次 |
| 批量分析任务 | 50 RP/次 |
| 高级仿真 | 100 RP/次 |
| 兑换 Tick 增量包 | 500 RP = 10 万 Tick |
| 兑换 Token 额度 | 100 RP = 10K Token |
| 兑换存储包 | 300 RP = 10 GB |
| 抵扣会员费 | 100 RP = $1（最高抵扣 50%） |

### 3.4 RP 生命周期

- **获取上限**：每日社区获取上限 2,000 RP
- **余额过期**：RP 余额在最后一次购买/获取后 12 个月过期
- **每日衰减**：余额每日被动消耗 0.1%，激励持续产出
- **消费记录**：所有变动记录可审计

## 4. 支付方式

| 支付渠道 | 支持币种 | 适用地区 | 支付方式 |
|---------|---------|---------|---------|
| Stripe | USD, CNY, EUR | 国际 | Visa, Mastercard, Alipay (海外) |
| Jeepay (支付宝) | CNY | 中国大陆 | 支付宝 App / 网页 |
| Jeepay (微信支付) | CNY | 中国大陆 | 微信支付 |

系统根据用户 IP 地理位置和浏览器语言自动推荐最优支付渠道，用户可手动切换。

## 5. 套餐升级与降级

| 操作 | 规则 |
|------|------|
| 免费 → 付费 | 立即生效，按比例收取剩余周期费用 |
| 升级（Pro → Team） | 立即生效，按比例补差价 |
| 降级（Team → Pro） | 当前周期结束后生效 |
| 取消订阅 | 当前周期结束前保持权限，之后降为免费版 |
| 退款 | 订阅 7 天内可全额退款（限首次） |

## 6. 专家认证体系

独立于付费等级，基于学术贡献与社区声誉：

| 认证等级 | 条件 | 权益 |
|----------|------|------|
| **领域贡献者** | 发布 3+ 公开模型包，获 50+ 星标 | 橙名标识，优先展示 |
| **认证专家** | 学术机构邮箱验证 + 论文 DOI 验证 | 蓝色认证徽章，审核权限 |
| **首席科学家** | 邀请制，平台官方认可 | 金色徽章，咨询委员会席位 |

## 8. 退款政策

| 场景 | 退款政策 | 处理时效 |
|------|----------|----------|
| 首次订阅 7 天内 | 全额退款 | 3-5 工作日 |
| 技术故障导致服务不可用 | 按比例退款 | 7 工作日 |
| 重复扣款 | 全额退还重复部分 | 1-3 工作日 |
| 用户误购（>7天） | 按比例退还未使用部分 | 7 工作日 |
| 欺诈争议 | 按支付渠道争议流程 | 30-90 天 |

**退款处理流程**：
1. 用户提交退款申请（通过管理控制台或邮件联系客服）
2. 系统验证退款资格（订阅时间、使用记录、退款历史）
3. 符合条件的申请自动审批；复杂情况（>7天、争议）转人工审核
4. 审批通过后，原路退回支付渠道
5. Stripe 退款即时到账；支付宝/微信退款 1-3 个工作日
6. 退款完成后，用户订阅立即终止，已消耗配额按实际使用扣除
